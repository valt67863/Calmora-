# **App Name**: Lovable Studio

## Core Features:

- AI Project Brainstormer: A generative AI tool that takes user prompts to suggest initial project ideas, structures, or content starting points for creative projects.
- Personalized Project Dashboard: A central user interface to view and organize all projects, with filters for 'All projects', 'Starred', 'Created by me', 'Shared with me', and 'Recents'.
- Project Content Editor: An intuitive interface for users to view, create, and edit the details, files, and assets of individual projects.
- Global Search and Filter: A unified search bar with keyword and advanced filtering options to quickly locate specific projects or resources within the platform.
- External Resource Manager: Functionality to seamlessly add, categorize, and link external web resources or files that are relevant to various projects.

## Style Guidelines:

- Color scheme: Dark. Inspired by vibrant digital art and creativity, evoking focus and innovation.
- Primary color: A rich, creative purple (#B34DE6) to signify inventiveness and depth.
- Background color: A very dark gray with a subtle purple tint (#201C22) for the primary interface and sidebar, ensuring content stands out.
- Accent color: A vibrant, clear blue (#5E8EDD) for interactive elements, highlights, and to complement the purple for dynamic contrasts.
- Body and headline font: 'Inter' (sans-serif) for its modern, highly legible, and objective aesthetic, suitable for a technical yet creative application.
- Utilize simple, elegant outline icons with consistent stroke weight to maintain a clean and professional look throughout the interface.
- Fixed left sidebar for navigation and primary categories, combined with a dynamic main content area focused on central user interaction and information display.
- Subtle and swift animations for navigation transitions, button hovers, and feedback on data interactions, enhancing the perception of responsiveness and fluidity.
