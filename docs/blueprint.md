# **App Name**: Enterprise Cloud Navigator

## Core Features:

- Headline and Value Proposition: A dynamically generated headline and subheadline emphasizing the cloud provider's value to enterprise businesses.
- AI-Driven Consultation Recommendation: Based on the user's role and company size (determined through a short survey), recommend the most appropriate consultation package, highlighting benefits using a tool. 
- Top Products Showcase: Dedicated section displaying top cloud products with concise descriptions, key benefits, and transparent pricing.
- Customer Testimonials: Carousel of rotating customer testimonials including company logos to build credibility.
- Consultation Request Form: Integrated form for users to quickly request a consultation, capturing key details for lead generation.
- Mobile Responsiveness: Fully responsive layout to ensure seamless user experience across all devices.

## Style Guidelines:

- Primary color: Deep sky blue (#30A2FF), symbolizing trust and innovation.
- Background color: Very light blue (#EBF4FF), offering a clean and professional backdrop.
- Accent color: Vivid blue (#0466C8), used for call-to-action buttons and highlights.
- Body and headline font: 'Inter', a sans-serif font providing a clean and modern appearance for both headlines and body text.
- Use clean, modern line icons to represent product features and benefits.
- Implement a grid-based layout with ample white space to enhance readability and visual appeal.
- Subtle animations, such as fade-in effects, for a polished and engaging user experience.
